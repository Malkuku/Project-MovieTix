普通用户可以进行的操作：
- 登录/注册
- 查询电影 --> 查询可放映场次
          --> 查询电影详情信息
- 选座购票
- 订单查询
- 订单支付
- 订单取消
- 订单退款
- 充值
- 个人信息查询
- 个人信息修改


普通用户购票流程:
用户登录/注册：验证身份，检查是否被拉黑/为管理员
->
查询可放映场次：选择场次并查询座位信息
->
选择座位：选择座位并确认
->
创建支付订单，添加座位信息(考虑预占用)，等待支付，创建支付记录表
->
用户支付/取消订单
->
支付：检查余额，不足则提示充值，充足则扣款，更新订单状态，更新支付记录表
取消：更新订单状态，批量删除座位，更新支付记录表
超时未支付：更新订单状态，批量删除座位，更新支付记录表

普通用户退款流程:
用户申请退款
->
管理员处理退款申请
->
更新退款申请表状态
->
更新订单状态
->
将资金返还给用户
